# Cover Prompt Presets

## Comparison Cover

适用: 工具对比、方案对比、A vs B

Prompt 要素:
- 双主体并列
- 明确对比焦点
- 简洁标题区
- 少量强调色
- 不要塞太多 logo 和小字

## Data Cover

适用: 研究总结、评测结果、趋势观察

Prompt 要素:
- 大数字或关键词
- 2-3 个核心视觉锚点
- 背景保持克制
- 数据感而非科幻感

## Personal Story Cover

适用: 经验复盘、成长、创作感受

Prompt 要素:
- 场景化主视觉
- 留白足够
- 标题不宜过长
- 优先情绪和氛围，而不是参数堆叠

## Output Rule

封面 prompt 至少说明:
- 目标平台
- 主题风格
- 主视觉元素
- 颜色方向
- 标题区要求
