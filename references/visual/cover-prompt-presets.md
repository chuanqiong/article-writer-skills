# Cover Prompt Presets (Nano Banana Pro)

## Comparison Cover

适用: 工具对比、方案对比、A vs B

Nano Banana Pro Prompt 模板:
```
Generate a split-screen comparison image showing [Subject A] on the left and [Subject B] on the right. [Action/interaction between them]. Use [lighting setup, e.g., 'studio lighting with soft shadows']. [Composition, e.g., 'centered composition with clear visual separation']. Style: [e.g., 'clean tech editorial, minimal color palette']
```

Prompt 要素:
- 双主体并列，明确左右分区
- 强动词开头 (Generate/Create)
- 具体描述光线和构图
- 少量强调色，避免堆叠 logo

## Data Cover

适用: 研究总结、评测结果、趋势观察

Nano Banana Pro Prompt 模板:
```
Create a data visualization cover featuring [main data point/number]. [Subject] arranged in [composition layout]. Lighting: [e.g., 'dramatic side lighting']. Camera: [e.g., 'low angle shot']. Style: [e.g., 'modern infographic aesthetic, muted teal and coral accents']
```

Prompt 要素:
- 大数字或关键词作为视觉锚点
- 2-3 个核心元素
- 背景保持克制
- 数据感而非科幻感

## Personal Story Cover

适用: 经验复盘、成长、创作感受

Nano Banana Pro Prompt 模板:
```
Generate an atmospheric scene depicting [emotional context/scene]. [Subject] in [location/context]. Camera: [e.g., 'low angle shot, shallow depth of field f/1.8']. Lighting: [e.g., 'golden hour backlighting']. Style: [e.g., 'cinematic, warm tones, film grain texture']
```

Prompt 要素:
- 场景化主视觉，强调情绪氛围
- 使用摄影术语控制相机 (low angle, shallow depth of field)
- 光线描述 (golden hour, softbox, dramatic)
- 优先情绪和氛围，而不是参数堆叠

## Output Rule

封面 prompt 必须包含:
- **强动词开头**: Generate/Create/Render/Design
- **Subject**: 具体描述主体
- **Composition**: 构图方式 (centered, rule of thirds, split-screen)
- **Camera/Lighting**: 摄影术语 (low angle, aerial, softbox, golden hour)
- **Style**: 风格描述 (clean-tech, magazine-editorial, cinematic)
- **Aspect ratio**: 16:9 (公众号封面) 或 1:1 (通用)

## Quick Reference

| Cover Type | 强动词 | 关键元素 |
|------------|--------|----------|

| Data | Create | data visualization, number, dramatic lighting |
| Personal Story | Generate | atmospheric, camera angle, golden hour |
