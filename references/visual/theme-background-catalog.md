# Theme And Background Catalog

## Theme Options

- `clean-tech`: 干净、理性、适合技术评测和工具文章
- `magazine-opinion`: 杂志感、适合观点和专栏
- `data-report`: 数据卡片感、适合对比和总结
- `creator-story`: 更有人味，适合个人经历
- `product-brief`: 像产品说明卡，适合方案和教程

## Background Options

- 纯色: 最稳，适合严肃内容
- 渐变: 适合科技感或趋势感
- 网格/坐标: 适合数据和方法论
- 纸张/杂志纹理: 适合专栏或观点文
- 卡片分区: 适合多要点总结和小绿书规划

## Selection Rules

- 技术评测优先 `clean-tech` 或 `data-report`
- 个人经验优先 `creator-story`
- 结构化报告优先 `product-brief`
- 如正文已经很密，背景要减法，不要再堆元素
