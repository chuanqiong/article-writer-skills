# Infographic Prompt Presets (Nano Banana Pro)

## Comparison Infographic

适用: 两种工具、两条路线、两种方案对比

Nano Banana Pro Prompt 模板:
```
Design a comparison infographic with [Subject A] and [Subject B] in a [horizontal/vertical] split layout. Render the text "[Title]" in [font style, e.g., 'bold sans-serif'] at the top. Include [comparison dimensions, e.g., 'price, speed, quality'] as visual anchors. Style: [e.g., 'clean data-report aesthetic with card-based sections']
```

Prompt 要素:
- 对比维度明确
- 左右或上下布局
- 文本用引号包围 + 字体描述
- 结论区 (可选)

## Process Infographic

适用: 工作流、方法步骤、执行路径

Nano Banana Pro Prompt 模板:
```
Create a step-by-step process diagram showing [workflow name]. Start with "[Step 1 label]" and end with "[Final step label]". Use numbered markers (1, 2, 3...) and directional arrows. Render each step as a [card/icon] with minimal text. Style: [e.g., 'product-brief aesthetic, consistent visual language']
```

Prompt 要素:
- 步骤编号 (1, 2, 3...)
- 起点 → 终点明确
- 每步 1 个动作
- 使用引号包围需要显示的文字
- 方向箭头连接

## Framework Infographic

适用: 观点框架、评估模型、能力地图

Nano Banana Pro Prompt 模板:
```
Design a framework diagram with [N] modules arranged in [hierarchical/circular/linear] structure. Render the title "[Framework Name]" in [font style]. Use arrows to show [relationships, e.g., 'dependencies', 'flow']. Each module contains one key concept. Style: [e.g., 'data-report aesthetic, muted color palette']
```

Prompt 要素:
- 模块分组明确
- 关系箭头或层级
- 每块 1 句说明
- 统一视觉语言

## Output Rule

信息图 prompt 必须包含:
- **强动词开头**: Design/Create/Render
- **图类型说明**: comparison/process/framework
- **核心结构**: 布局方式 (split, linear, hierarchical, circular)
- **文本渲染**: 用引号包围需要显示的文字，描述字体 (bold sans-serif, elegant serif)
- **视觉锚点**: 图标、数字、箭头等
- **风格一致性**: 统一的视觉语言

## Text Rendering Tips

Nano Banana Pro 文本渲染最佳实践:
1. **用引号包围文字**: `"URBAN EXPLORER"` 而不是 URBAN EXPLORER
2. **描述字体风格**: `bold sans-serif`, `elegant serif`, `handwritten script`
3. **指定字号层级**: `title in 24px`, `subtitle in 16px`
4. **多语言支持**: 可指定翻译目标语言
