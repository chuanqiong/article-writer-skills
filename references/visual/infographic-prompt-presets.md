# Infographic Prompt Presets

## Comparison Infographic

适用: 两种工具、两条路线、两种方案对比

必含:
- 对比维度
- 左右或上下布局
- 结论区
- 尽量避免大段文字

## Process Infographic

适用: 工作流、方法步骤、执行路径

必含:
- 步骤编号
- 起点终点
- 每步 1 个动作
- 风险或注意事项标注

## Framework Infographic

适用: 观点框架、评估模型、能力地图

必含:
- 模块分组
- 关系箭头或层级
- 每块 1 句说明
- 统一视觉语言

## Output Rule

信息图 prompt 至少说明:
- 图类型
- 核心结构
- 目标阅读场景
- 文案密度控制
- 是否要适配微信长图或小绿书卡片
