# Colloquial Replacements

## Common Replacements

| Formal | More Natural | Note |
| --- | --- | --- |
| 进行分析 | 分析一下 | 动词回归口语 |
| 进行对比 | 对比一下 | 去掉名词化 |
| 实现提升 | 提高了 | 直接说结果 |
| 具备优势 | 更占便宜 | 可按文风调整 |
| 值得注意的是 | 注意一点 | 很多时候直接删掉更好 |
| 需要指出的是 | 先说结论 | 这类铺垫常可删除 |
| 由此可见 | 所以 | 若前文已充分可直接下结论 |
| 总的来说 | 我的结论是 | 或直接给结论 |
| 综上所述 | 最后说结论 | 结尾避免公文腔 |
| 在某种程度上 | 某些情况下 | 具体化范围 |
| 具有一定的 | 有点 | 看语境选择力度 |
| 相对而言 | 相比之下 | 更像自然比较 |
| 从本质上看 | 说到底 | 观点文更自然 |
| 用户体验 | 用起来的感觉 | 体验文优先具体表达 |
| 高效地 | 更快地 | 能量化更好 |
| 显著提升 | 提高了很多 | 最好补数据 |
| 开展工作 | 开始做 | 去掉公文味 |
| 获取信息 | 找资料 | 口语化 |
| 给予支持 | 支持 | 删冗词 |
| 产生影响 | 有影响 | 或写具体影响结果 |

## Principles

- 口语化不等于随便，优先自然、具体、可读。
- 如果换成口语后仍然空泛，问题不在词，而在内容。
- 技术文不必强行口语化到失真，准确优先，僵硬次之。
